# 02_fe_react/src

**책임:** React 앱 소스. 현재는 결과지 단일 페이지 프로토타입 단계 — 검증 가설은 *"INFP 결과지 한 장이 진짜 공감되는가"*.

## 진입점

- [main.tsx](main.tsx) — React 18 루트 마운트 (`StrictMode`).
- [App.tsx](App.tsx) — 라우팅 없이 단일 결과 컴포넌트 렌더 (의도적).

## 비자명 제약

- 전역 스타일은 [styles/global.css](styles/global.css) **한 곳에만** 둔다. 컴포넌트 스타일은 전부 같은 폴더의 `.module.css`.
- **App.tsx에 라우터가 의도적으로 없다.** 결과지 카피 톤 검증이 1순위라 라우팅을 미뤄둠. 테스트 입력 페이지를 붙이는 시점이 라우터 도입 시점.
- 컴포넌트 / 디렉터리 구조 결정은 *결과지 검증이 끝난 뒤* — 그 전에 추상화 들어가지 말 것.
