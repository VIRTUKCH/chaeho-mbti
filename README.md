## 한 줄 정체성

**기존 MBTI가 "공감되지 않는다"는 문제를 정조준한, 공감되는 한국식 성격 테스트.**

기존 MBTI는 칼 융 심리학에 뿌리를 둔 학술적 분류라, J/P 차이를 따로 공부해야 알 정도로 일상 직관과 멀다. 그래서 결과를 받아도 "정말 내가 이게 맞나?" 싶고, 친구에게 보여주면 "너가 I라고?"라는 반응이 돌아오기 일쑤다. 우리는 학술적 정확성 대신 **"느낌이 맞다"**는 공감을 목표로 한다.

## 페르소나 — 누가 이걸 좋아할까

인구통계(나이/성별/직업)로 좁히지 않는다. 대신 **콘텐츠/기존 MBTI와의 관계**로 정의한다.

### 1. 결과가 "내 얘기"여야 만족하는 사람들

- 자기 분석·자기 이해에 관심이 많다
- "어떻게 이걸 알았지?" 싶은 디테일에 반응한다
- 결과 페이지를 캡처해서 카톡/인스타에 올리는 게 자연스럽다

### 2. 기존 MBTI에 미묘한 불만이 있는 사람들

- "내가 정말 ENFP가 맞나?" 하고 결과를 의심해본 적이 있다
- "J랑 P는 도대체 뭔 차이야?" 식으로 카테고리 자체에 답답함을 느낀다
- 주변에서 "너가 I라고?", "너가 T라고?" 같은 의외라는 반응을 들어본 적이 있다
- 결과 글이 너무 길고 일반론이라 끝까지 안 읽어본 적이 있다

### 3. 콘텐츠를 "화제거리"로 쓰고 싶은 사람들

- 회식·소개팅·단톡방에서 분위기 띄우는 도구로 쓴다
- 친구 결과와 비교하면서 노는 게 본 목적이다
- "이거 딱 너야", "이거 우리 팀장이야" 식으로 태깅하고 싶다
