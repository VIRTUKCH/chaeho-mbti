- [00_overview.md](00_overview.md) — 4카드 × 후보 이론 매트릭스 (리서치 대기).
- [01_연애/](01_연애/) — Sternberg TLS 등 연애 카드 후보 이론.
- [02_가족/](02_가족/) — Adler 출생순위 등 가족 카드 후보 이론.

## 비자명 제약

- **이 디렉터리는 후보 풀이지 확정 매핑이 아니다.** README 원칙 "학술 매핑은 카피 박힌 다음 역방향으로"에 따라, 여기 적힌 이론은 카드 결과 카피가 박힌 다음 역방향으로 선택된다.
- **카드 순서는 README `CARD_ORDER` 기준** (`love > family > friend > work`). 다른 곳에서 임의 재정렬 금지.
- **사용자 화면에는 학술 어휘 0.** 여기 이론·논문·척도 약어는 백엔드 안전망이고, 카드 결과 카피·문항·라벨에는 노출되지 않는다.
- **여러 카드에 걸치는 이론**(attachment 등)은 각 카드 폴더에서 *그 카드 관점*으로 재해석한 파일로 둔다. 캐논 1개 + 링크 방식은 피한다 — 카드별 응집을 우선.
