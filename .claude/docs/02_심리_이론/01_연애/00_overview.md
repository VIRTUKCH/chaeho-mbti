# 01_연애 — 후보 이론 비교

> 연애 카드 백엔드 풀. 사용자 화면에는 학술 어휘 0. 카드당 4~6문항으로 신호를 잡아야 하고, "공감되지 않는 결과"를 피하려면 **일상 맥락 변수**(친밀·열정·헌신, 회피·불안 등)에 닿아야 한다.

## 진단 차원 후보

연애 카드가 잡을 수 있는 일상 변수:

1. **사랑의 구성 요소** — 친밀(intimacy) / 열정(passion) / 헌신(commitment) 의 균형
2. **애착 패턴** — 회피(avoidance) / 불안(anxiety) 2축
3. **사랑 스타일** — 표현 방식의 색채(낭만/우정/유희/실리/소유/헌신)
4. **관계 유지 동력** — 만족·대안·투자 (장기관계 평가에 치우침)

카드 4~6문항 제약을 고려하면 1~2축, 많아야 3축이 한계.

## 잠정 추천 — **Sternberg 사랑의 삼각형 (TLS-15)**

평가축 4개로 본 이유:

1. **카드 적합성** — 친밀·열정·헌신 3축은 솔로/연애 중/장기관계 모두에 응답 가능. 임상 톤이 약해 모자이크 카드로 가볍게 입길 수 있음.
2. **매핑 난이도** — 3축 고/저 조합이 곧 Sternberg의 7~8개 학술 유형. 임의 컷 없이 유형명을 그대로 가져올 수 있음(Sternberg, 1986).
3. **한국 사용성** — TLS-15는 Kowal et al.(2024) 37개 언어 교차 타당, 5점 척도 15문항이라 카드당 4~6문항으로 패션·헌신·친밀 각 1~2문항씩 잘라 쓸 수 있음.
4. **학술 신뢰도** — TLS-15와 TLS-45 모두 KMSS·Rubin Love Scale과 수렴타당. 메타분석급 합의 존재.

**보조축으로 ECR-R 회피/불안 1축을 얹는 것**도 고려 가치 있음 — Sternberg가 못 잡는 "왜 가까워지면 도망가나" 통증을 메우는 변수. 단 카드 4~6문항 안에 끼우려면 회피만 2문항 정도.

## 유형 후보 (Sternberg 8유형, 학술명 그대로)

친밀·열정·헌신의 고(H)/저(L) 조합:

1. **Non-love** (L·L·L) — 셋 다 부재 (모자이크에선 컷 후보)
2. **Liking / Friendship** (H·L·L) — 친밀만
3. **Infatuated Love** (L·H·L) — 열정만, 첫눈에 반함
4. **Empty Love** (L·L·H) — 헌신만, 식은 장기관계
5. **Romantic Love** (H·H·L) — 친밀+열정, 헌신 미확정
6. **Companionate Love** (H·L·H) — 친밀+헌신, 식은 동반자
7. **Fatuous Love** (L·H·H) — 열정+헌신, 친밀 없는 속도전
8. **Consummate Love** (H·H·H) — 셋 다, 완성형

카드용으로 **Non-love는 컷**, 나머지 7개로 가는 게 현실적. 더 줄이면 Liking·Infatuated·Empty·Romantic·Companionate·Consummate 6개도 무난.

## 버린 옵션과 이유

- **Chapman 사랑의 언어** — 학술 타당도 결정적으로 약함. 백엔드 기둥으로 쓰면 "공감 안 됨" 비난을 그대로 떠안음. 카피 어휘 차용만 OK.
- **Hatfield PLS / Hendrick RAS** — 단일축이라 유형 모자이크 불가. 보조 점수용으로만 가치.
- **Rusbult Investment Model** — "현재 연인 있음" 전제가 강함. 솔로·짝사랑 응답자 비율 높은 트래픽형 테스트에 부적합.
- **Lee LAS** — 6유형이 매력적이지만 요인 변별이 흐리고 한국 타당화 빈약. Sternberg 유형 라벨의 카피 톤 보강용으로만 차용 가능.
- **ECR-R 단독** — 임상색이 너무 짙어 "성격 진단받는 느낌". 단 Sternberg 위에 회피 1축 얹는 하이브리드는 매력 있음.
