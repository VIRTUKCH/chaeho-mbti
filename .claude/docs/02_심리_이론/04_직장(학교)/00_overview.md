# 04_직장(학교) — 후보 이론 비교

> 4카드 모자이크 중 **직장(학교) 카드** 백엔드용 학술 이론 풀. 사용자 화면 어휘 0, 백엔드 전용. 카드 컨셉은 "조직(직장+학교) 안에서의 행동 스타일" — 회의 발언·과제/업무 추진·갈등 대응·동기 원천을 4~6문항으로 잡는다. 연애 카드(Sternberg TLS-15 확정)와 카논이 겹치지 않아야 한다.

## 진단 차원 후보

이 카드가 잡아야 할 일상 신호:

1. **회의/수업 발언량과 톤** — 주도하나 / 묵묵 / 분위기 / 관찰
2. **과제·업무 추진 방식** — 계획·완수형 vs 즉흥·반응형
3. **갈등 발생 시 디폴트 반응** — 정면/회피/타협/협력/수용
4. **동기 원천** — 자율성·유능감·관계성, 또는 권력·지위·인정

이 4축이 하나의 이론에 다 들어가는 캐논은 없다. **하이브리드가 현실적**.

## 잠정 추천 + 이유

**추천: Big Five 직무 facet + HEXACO 정직-겸손 하이브리드** (위 표 마지막 행).

이유:

- **학술 신뢰도가 다른 후보를 압도한다.** Barrick & Mount(1991) 이후 Conscientiousness는 거의 모든 직군에서 직무수행을 예측하고, HEXACO H는 Big Five가 못 잡는 CWB 분산을 한국 표본에서 추가로 설명한다.
- **카드가 잡아야 할 4축에 모두 닿는다.** Conscientiousness facet(추진·완수), Extraversion facet(회의 발언·주도), HEXACO H(공정성·갈등 디폴트). 동기 축은 약하지만 facet 라벨링으로 보완 가능.
- **한국 타당화가 둘 다 완비.** K-BFI-15·K-BFI-2와 K-HEXACO-PI 한국판 모두 KCI 게재. 사용자 화면엔 라벨 0이지만, 백엔드 안전망으로는 가장 단단함.
- **DiSC 트레이드오프 평가:** 친숙함은 매력적이지만, 본 프로젝트의 정조준 대상이 "MBTI류 학술적으로 약한 4유형 단순화"다. DiSC를 백엔드에 깔면 같은 약점을 답습한다. 사용자에게 노출되는 카드 카피·유형 라벨은 DiSC식 친숙한 톤(예: "주도형/조율형/꾸준형/관찰형")으로 만들되, **백엔드 매핑·문항 설계는 Big Five facet + HEXACO H로** 가는 게 정직하다.

## 유형 후보 (5~8개, 학술명 그대로)

추천 이론에서 파생되는 facet 조합 — 사용자 라벨이 아니라 백엔드 코드용:

1. **High-Industriousness / Low-Assertiveness** — 묵묵 추진형
2. **High-Assertiveness / High-Industriousness** — 주도 추진형
3. **High-Activity / Moderate-Industriousness** — 분위기 메이커형
4. **High-Intellect(Openness) / Low-Assertiveness** — 관찰·아이디어형
5. **High-Honesty-Humility / High-Orderliness** — 원칙·공정형
6. **Low-Honesty-Humility / High-Assertiveness** — 전략·정치형(어두운 변형)
7. **High-Sociability(E facet) / Low-Industriousness** — 관계 우선형
8. **Moderate 전반 / High-Adaptive(낮은 Orderliness + 높은 Openness)** — 유연 적응형

카드 결과 카피 박힐 때 6~7개로 추려서 모자이크화. 음수 facet 유형(6번)은 "공감되지만 부정적"이라 카피 톤 신중.

## 버린 옵션과 이유

- **MBTI 직무 응용** — 본 프로젝트가 정조준해 까는 대상. 백엔드로 쓰면 자기모순.
- **Holland RIASEC** — 학술은 견고하나 "직업 흥미"라 카드 컨셉("조직 안 행동")과 결이 다름. 학교 카드 적합도도 낮음(고등학생까지 직업 미정).
- **Reiss 16 Motivation Profile** — 한국 타당화 부재 + 16축 압축 임의성 + 저자 자가출판 비중. 동기 깊이는 매력적이나 백엔드 안전망 역할 못 함.
- **Belbin Team Roles** — 한국 타당화 거의 없음, 9역할 SPI 자체의 변별타당도 비판, 학교 카드 미커버.
- **TKI 단독** — 5스타일 캐논은 매력적이나 갈등 1축만 잡음. 회의 발언·추진·동기 못 잡아 카드 1개로는 부족. 단, 추천안의 문항 1~2개로 갈등 시나리오 차용은 가능.
- **SDT 단독** — 학술·한국 검증은 최강급이지만 3 욕구로 5~8 유형 모자이크 못 뽑음. 추천안의 동기 축 보강용 인용 정도로만.
- **DiSC 단독** — 친숙함은 압도적이지만 학술 약함이 본 프로젝트 차별화와 충돌. 사용자 라벨 톤만 차용하고 백엔드는 추천안으로 가는 게 정직.
