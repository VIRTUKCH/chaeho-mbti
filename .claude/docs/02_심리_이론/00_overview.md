# 02_심리_이론 — 리서치 계획 & 4카드 매트릭스

> 4카드(연애·가족·친구·직장) 각각의 **검사 차원**과 **잠정 N개 유형**의 학술 백엔드를 채우는 자리. 이 파일은 *리서치 계획*이고, 결과는 각 카드 폴더의 `00_overview.md` 와 `0X_<이론>.md` 에 들어간다.

## 목적

- 카드별로 *진단 가능한* 차원을 정의한다. (감으로 만들지 않는다)
- 각 차원을 뒷받침할 후보 이론·척도를 카탈로그한다.
- 후보 중 하나를 잠정 추천하고, 거기서 파생되는 유형 후보 5~8개를 뽑는다.
- **결과 카피가 박힌 다음**, 역방향으로 이 카탈로그에서 확정 매핑을 고른다 (README 원칙).

## 카드별 산출물 (각 카드 폴더 `00_overview.md` 에 채워짐)

1. **차원 정의** — 이 카드가 진단하려는 심리 변수 1~3개. 한 문장씩.
2. **후보 이론 비교표** — 3~5개 후보. 컬럼: 이론 / 차원 / 검사 도구 / 한국 검증 / 매핑 난이도 / 트레이드오프.
3. **잠정 추천 + 이유** — 평가축(아래) 기준으로 1개 선택.
4. **유형 후보 5~8개** — 잠정 추천 이론에서 파생. 학술명 그대로 OK (카피팀이 한국어 밈으로 변환).
5. **버린 옵션과 이유** — 왜 다른 후보를 떨어뜨렸는지 한 줄씩.

## 공통 평가축

| 축 | 무게 | 메모 |
|---|---|---|
| 학술 신뢰도 | 높음 | 메타분석·합의된 척도 존재 여부 |
| 한국 검증 | 높음 | 한국어판 척도·한국 표본 연구 유무 |
| 차원 → N개 유형 매핑 난이도 | 중 | 연속 차원을 N개 칸으로 자를 때 임의성 |
| 카드 컨셉 적합성 | 높음 | 그 카드의 *일상 맥락*에 닿는가 |
| 응답 부담 | 중 | 카드당 4~6문항 안에 신호 잡히는가 |

## 카드 간 선정 원칙

- **같은 이론을 여러 카드에 캐논으로 쓰지 않는다.** 결과지 4장이 비슷해 보이면 모자이크가 무너진다.
- 빅파이브·HEXACO 같은 횡단 이론은 *측면(facet)* 만 발췌해 한 카드에 묶는 건 OK.
- 한 이론이 여러 카드에 걸치면(예: 애착), **카드 관점으로 재해석한 별도 파일**로 둔다. 캐논 1개 + 링크 방식 금지.
- 카드 순서는 README `CARD_ORDER` (`love > family > friend > work`) 고정.

## 면책 / 제약

- 이 폴더의 이론은 **후보 풀**이지 확정 매핑이 아니다.
- 사용자 화면에 학술 어휘 0. 여기 적힌 척도 약어·논문 인용은 백엔드 안전망일 뿐.
- 한국 검증이 없는 이론도 후보에 넣되 *플래그*. (한국 표본 부재가 곧 탈락은 아님)

| 카드 | 잠정 차원 | 검토한 후보 | 잠정 추천 | 유형 수 |
|---|---|---|---|---|
| 01_연애 | 친밀·열정·헌신 (3축) | Sternberg TLS, ECR-R, Hatfield PLS, Lee LAS, Rusbult IMS, Hendrick RAS, Chapman 5L | **Sternberg TLS-15** (Kowal et al. 2024, 37개 언어) | 6~7 (Non-love 컷) |
| 02_가족 | 응집(cohesion) × 유연성(flexibility) | Adler 출생순위, FACES-IV, Bowen DSI, Baumrind 양육, ECR-RS 부모애착, Schwartz, FS-K 가족주의, ROCI-II | **FACES-IV (Olson Circumplex)** + Bowen DSI-SF 보조 (K-FACES 이민숙 2014) | 6 |
| 03_친구 | 공감 4차원 (PT·EC·PD·Fantasy) | K-EQ, Davis IRI, BFI-2 측면, HEXACO H+X, Schwartz PVQ, Aristotle 3유형, Dunbar, Jourard SDS | **Davis IRI / K-IRI** (강일 외 2009, α=.80) | 6+1 fallback |
| 04_직장(학교) | 성실성·외향성 측면 + 정직-겸손 | DiSC, MBTI 직무판, Holland RIASEC, Reiss 16, Belbin, TKI, SDT, BFI-2 직무 facet, HEXACO | **BFI-2 직무 facet + HEXACO H 하이브리드** (K-BFI-2 / K-HEXACO-PI) | 8 |

> 각 카드의 비교표·잠정 추천 근거·유형 후보 학술명·버린 옵션 상세는 카드 폴더의 `00_overview.md` 참조.

### 크로스 카드 점검 (캐논 중복 금지 원칙)

- **Sternberg** → 연애 캐논. 다른 카드 사용 금지.
- **FACES-IV** → 가족 캐논. 다른 카드 사용 금지.
- **Davis IRI / 공감** → 친구 캐논. 다른 카드 사용 금지.
- **BFI-2 / HEXACO** → 직장 캐논. 친구 카드는 후보에 있었으나 IRI로 확정.
- **애착(ECR-R 계열)** → 어느 카드의 캐논도 아님. 연애 카드의 *보조 1축*(회피 2문항)으로만 옵션. 가족 카드의 ECR-RS는 라벨 충돌로 떨어뜨림.
