## 한 줄 핵심 답변

**chaeho-mbti의 backbone 선택은 "학술 신뢰도와 viral 친화성의 trade-off"라는 본질을 받아들이고 시작해야 한다.** 학술 ★4~5짜리(Big Five, HEXACO, Schwartz)는 연속 스펙트럼이라 카드화가 어렵고, viral ★★★★+짜리(Enneagram, 12 Archetypes, 출생순위, 5 Love Languages)는 학술적으로 깨졌거나 약하다. 양쪽을 모두 만족시키는 "하이브리드 backbone"은 가능성보다 환상에 가깝다 — 카카오 마음날씨 Big5가 8년 610만 누적해도 viral 폭발 못 한 게 그 증거. 그러므로 **backbone은 하나의 이론을 정통으로 선언하는 게 아니라, 여러 이론에서 분류 디테일·카피 디테일·검증 도구를 모듈로 빼와서 자체 IP를 짜는 게 맞다.**

**확신도: ★★★★☆ (4/5)** — 트레이드오프는 카카오 8년 실증으로 명확하나, 최선의 모듈 조합은 베타 사용자 테스트 없이는 단정 불가.

## chaeho-mbti backbone 전략 권장 (라운드 2.5 반영)

### 추천 1순위 — 모듈 조합 전략 (자체 IP 빌드)

단일 이론 backbone 대신, 아래 모듈을 결합:

- **분류 뼈대(N개 카드)**: VIA/CST 24강점 또는 Sternberg 8유형 (학술 ★★★★ + 카드화 명확)
- **공감 차원 보조**: EQ (T발놈 영역 학술화), Attachment 2차원 (연애 결과지 보조)
- **한국적 정체성 차원**: 주체성-대상성-자율성 (K-장녀 같은 한국 특화 라벨 정당화)
- **카피 영감 소스**: 12 Archetypes (캐릭터 디테일), Enneagram 동기·두려움 (서사 디테일), Birth Order (K-맥락 라벨)
- **명시 회피**: HEXACO H 단독 라벨(도덕 라벨링 위험), 5LL primary language 가설(학술 기각됨)

이 조합의 핵심: **이론 이름을 마케팅 메시지로 노출하지 않는다.** 학술은 백엔드 검증·카피 디테일 소스로만. 표면은 "재미로 보는" + 공감 카피.

### 추천 2순위 — 단일 이론에 올인해야 한다면

- **viral 우선**: Attachment 4유형 (K-ECR 학술 + 연애 viral 검증). 단 Fraley & Waller 2015에서 categorical 아닌 dimensional이 학계 정설이라 4유형 카드화는 학술 사기 리스크.
- **학술 우선**: VIA/CST 24강점 (학술 ★★★★ + 한국 표준화 완료). 단 24장 카드는 16유형보다 인지 부담 큼.
- **블루오션**: Sternberg 8유형 (37개 언어 cross-cultural 검증된 학술 + 8유형 카드 viral 미개척).

### 절대 피해야 할 선택

- "융의 12 archetype 기반"이라는 마케팅 문구 — 학술 사기. 융은 12로 한정한 적 없음.
- "공감지수 낮음 = T발놈" 식 EQ 부정 라벨 단독 사용 — 자기서빙 편향 + 차별 정서 위험.
- HEXACO H 점수를 정직-거짓말쟁이 축으로 직접 카드화 — 수협은행 MBTI 채용 차별 같은 무기화 위험.
- 4유형 분류(DISC, 4기질, 혈액형, 사상체질) 단독 — viral 분기 약함 + 식상감.

## 라운드 2.5 핵심 반박 요약

전체 반박은 각 이론 폴더에 분산 배치. 종합하면:

1. **학술 backbone = viral 차별화** 가설 자체 의심 (강) — 카카오 Big5 8년 6.1M 실증
2. **HEXACO H = 부정 도덕 라벨링 위험** (강) — BuzzFeed "결과는 항상 긍정" 법칙 위반
3. **12 Archetypes는 마케팅계도 "Top10 Bullsh*t #2"** (강) — Jung 정통성 부재
4. **Attachment 4유형 카드화 = 학술 사기** (강) — 학계는 2차원 정설
5. **TikTok 애착 self-diagnosis 청년 정신건강 피해** (중) — viral의 그림자
6. **에니어그램은 한국 신비주의·종교 이미지 부담** (중~강) — 9유형 분해 불가
7. **"학술 + viral 패키징"은 trade-off** (강) — 학술색 강조 시 "재미로 보세요" 면책 깨짐

## 이 리서치의 맹점

- **한국 viral 정량 데이터 부재**: 카톡 공유율·인스타 해시태그 수 같은 정량은 못 측정. 정성 사례 의존.
- **5LL meta-analysis 원문 paywall**: 2024 Impett 논문 본문 미접근, abstract+요약 의존.
- **Reiss 한국 사용 사례 미확보**: 라이선스 모델이라 공개 검색 빈약. 컨설팅 영역에 묻혀 있을 가능성.
- **시간 민감성**: 학술은 안정적이나 한국 viral 트렌드는 6개월 단위 변동. 에겐-테토·SBTI 다음 분류 6개월 내 출현 가능.
- **베타 사용자 테스트 미수행**: 어떤 이론 모듈이 한국 사용자에게 가장 공감되는지는 N=20~30 베타 테스트 없이는 단정 불가. 본 리서치는 사전 큐레이션일 뿐.
