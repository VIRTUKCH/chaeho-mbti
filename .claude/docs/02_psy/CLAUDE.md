## 비자명 제약

- **결과지 카피의 근거가 되어야 한다.** 학술 논문 요약이 아니라 *"이 분류가 왜 공감되는가 / 왜 안 되는가"*의 설명. 각 이론 폴더의 `00_overview.md` 끝에는 **chaeho-mbti 결과지 카피 번역 예시 1~2개**가 의무.
- **"MBTI는 비과학적"이라는 일반 비판은 정리하지 않는다** — 우리가 이미 그걸 전제로 학술 정확성 대신 공감을 택했기 때문.
- **각 이론은 7개 평가 기준으로 채점된다**: 분류 명확성 / 공감 친화성 / 밈·캐릭터화 잠재력 / 학술 신뢰도 / MBTI 차별화 / 한국 시장 적합성 / viral 적용 사례.
- **악마의 변호인 절을 의무 포함**한다. 단일 이론 신봉 금지. 카카오 마음날씨 Big5가 8년 6.1M 누적해도 폭발 viral 못 했다는 것 — "학술 backbone = 차별화" 가설 자체가 의심받는다.
- [04_viral/03_checklist.md](../04_viral/03_checklist.md)와 함께 읽어야 의미가 있다. 이론 선택만으로는 viral 보장 X — 카피 톤과 톤 가드레일이 결합돼야 함.
