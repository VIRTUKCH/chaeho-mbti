# 01. 여태 어떤 심리/성격 컨텐츠들이 있었나

> **목적**: 2015~2026년 한국·글로벌에서 흥했던 심리·성격 테스트/컨텐츠의 카탈로그. 다음 단계(02 강점, 03 약점, 04 저격 원리, 05 다음 갈증)의 입력 자료.
>
> **범위**: 정식 심리검사 + 글로벌 사이트 + 한국 사이트/플랫폼 + 한국 바이럴 짤테스트(시기별) + 아시아 비교 + 비즈니스 모델.

## 1. 정식 심리검사

### MBTI (Myers-Briggs Type Indicator)
- Briggs·Myers 모녀가 융 이론 기반으로 2차 대전기 개발. 한국 도입 1990년대(한국MBTI연구소), **2018년 구글 트렌드 한국 검색량 1위 고착**, **2020.06.13 MBC '놀면 뭐하니?' 방영 직후 검색량 정점**이 결정적 분기점.
- 2021.12 한국리서치: 한국인 절반 이상 경험, **19~28세 90%+ 경험**.
- 실제 한국에서 응시되는 것의 대다수는 정식 KPTI(유료)가 아니라 **16personalities**.

### Big Five (OCEAN / FFM)
- 1961 Tupes·Christal → 1980s Goldberg 명명, Costa·McCrae NEO-PI(1985)/NEO-PI-R(1992) 측정 도구. 학계 정식 표준이나 **한국 일반 대중 인지도 낮음** — "신뢰성 있는 모델" 광고용 자산으로만 유용.

### 에니어그램
- 한국 도입 1984(박인재 번역) → 1997 우재현 협회 → 1998 윤운성 학회 → 2001 **한국형 KEPTI 표준화**. 1990s 후반부터 종교계·기업 워크숍 중심으로 확산. 누적 수강자 5만+.

### DISC
- 1928 William Marston(거짓말 탐지기 발명자) 정서·행동 이론. 한국 직장 워크숍 표준 도구. **k-DISC**(네오아이즈)가 한국인 1,000명 재직자 표준화.

### HEXACO
- 2004 Ashton·Lee, 다국어 어휘 연구로 도출. Big Five + **정직-겸손(H)** 차원 추가. 학계에서는 빠르게 채택, 대중 인지도는 낮음.

### MMPI / MMPI-2
- 1940s Hathaway·McKinley(미네소타대). **MMPI-2 한국판 2005년**(번역 한경희·임지영, (주)마음사랑 단독 판매). 정신과·상담센터 임상 표준 — 일반 소비자용 아님.

## 2. 글로벌 웹 기반 성격 테스트 사이트

### 16personalities (16personalities.com)
- **NERIS Analytics Limited(영국 캠브리지) 2011 설립**(창업자 Stikoniene·Stikonas). MBTI 4지표·16유형 차용하되 실제 산정 모델은 Big Five 기반의 **NERIS Type Explorer**(공인 저널 게재 없음).
- 무료 60문항, 45개 언어, **누적 응시 10억회+**. 한국에서 사실상 "MBTI 검사 = 16personalities"로 통용.
- 결과 = **4글자 + 캐릭터 일러스트 + 강점/약점/연애/우정/커리어/자녀양육 카드** → 짤테스트의 시각 패키징 표준.

### Truity (truity.com)
- **Molly Owens(상담심리 석사)가 2012 설립**(캘리포니아). "기업용 고가 vs 무료 장난" 갭 메우는 포지셔닝.
- **일 약 10만 방문, 누적 6,000만+ 응시**, Inc. 500 빠르게 성장하는 회사 383위. Big Five·MBTI류·에니어그램·DISC 모두 보유.

### CareerExplorer (careerexplorer.com, 구 Sokanu)
- 2012~ 운영. 20~30분, 5섹션(흥미·성격·작업환경·이력·페이스). Big Five + Holland Code + O*NET 직업 DB 1,500개+ 결합. 미국 대학 진로상담실 표준 링크.

### BuzzFeed Personality Quizzes
- 2014~ 폭발. Kissmetrics 데이터: 퀴즈 평균 페이스북 공유 **51,968회 vs 일반 기사 15,527회**(3.3배). 자기-관련성이 메달프리프론탈코르텍스 보상 회로 자극(Psychology Today 분석).
- 현재는 **IP 캐릭터 매칭**("Which character are you")이 주력 — 신규 영화/드라마 출시마다 퀴즈 발행으로 트래픽 흡수.

### Uquiz / Heywise / Playbuzz (영어권 짤퀴즈 플랫폼)
- **Uquiz**: 유저 생성 퀴즈, 십대~Z세대 트위터에서 미적 감성 퀴즈("which aesthetic are you")로 강함.
- **Heywise**: 사내 제작 only, UGC 없음. 트리비아 + 캐릭터 매칭.
- **Playbuzz**: 퍼블리셔/유저 양쪽에 임베드 가능한 퀴즈 빌더. **결과 공유 → 친구도 도전**의 viral loop를 인프라화한 사례.

## 3. 한국 웹 기반 성격 테스트 사이트/플랫폼

### ktestone (ktestone.com)
- 한·영·중·일·스페인어 다국어. "퍼스널컬러", "연애 캐릭터", "한글 표 성격", "스마일 컬러" 등 자체 IP + 사주 풀이 결합. **사실상 개인/소규모 운영체**(GitHub Pages 서브 테스트 다수). 수익화는 광고+사주 결제 추정 — 비공개.

### 푸망 (poomang.com)
- 운영사 **(주)푸른망아지**, 창업자 류승준, **2019 하반기 창업 / 2020.12 런칭**, 직원 약 20명.
- **누적 사용자 1,000만+, 누적 테스트 플레이 7,500만+, MAU 200만, 월 매출 1억(2021)**.
- 수익 모델: **기업 콜라보 테스트 영업**(넷플릭스·SM·디즈니·CJ ENM 등). 광고 AE 직군 별도 채용 — B2B 영업이 매출 핵심.

### 방구석연구소 (banggooso.com)
- 운영사 **더에스엠씨그룹(The SMC Group)** — 김용태 대표, 2009 창립 디지털 콘텐츠 솔루션 기업. 12개 사내 콘텐츠 연구소 중 하나.
- "연애 능력 테스트" 공개 5일 만에 200만 회. 12유형·13질문 표준 포맷.
- 수익 모델: **Ad ID 마케팅 솔루션** — 유저 소셜 프로필 + 짤테스트 결과 데이터 묶어 광고주 타게팅 자산 판매. 단순 광고 노출이 아닌 **데이터 기반 콜라보 제작 + 마케팅 솔루션** 패키지.

### 테스트모아 (testmoa.com)
- "심리테스트 No.1" 표방. 성격/관계/능력/트렌드/전문검사/상식퀴즈 6 카테고리 허브. DISC·에겐테토 등 트렌드 즉각 패키지화.

### 테스트잇 (test-it.co.kr) / 메타브 (metavv.com) / 타입스 (types.my) / 테스트하로 / 테스트모리 / 심콩
- 짤테스트 인터랙티브 플랫폼 그룹. 타입스는 **에겐테토 테스트 2025-05-23 기준 714,250명 응시**.

### 동물상 테스트 (animalface.site)
- 유튜버 **조코딩** 제작. 강아지·고양이·여우·사슴·토끼·곰·공룡 등으로 분류, 연예인 사진 학습. **앱 누적 470만 뷰**. 회원가입 없이 클라이언트 처리.

### 한국MBTI심리연구소 (kmbti.co.kr) / 한국가이던스 / 마음사랑
- 정식 라이센스 보유 기관. 무료 간이검사 페이지는 SEO 노출되나 정식 검사는 유료.

## 4. 한국 바이럴 짤테스트 (시기별)

### 2017~2019 — 발화기
- **퍼스널컬러 K-Beauty 마케팅 진입**(2017 에뛰드하우스 명동 플래그십 퍼스널컬러 스튜디오) → 자기 진단 컨텐츠 1차 토대.
- **MBTI 온라인 발화**(2017~2019 트위터·디시·블로그 중심), 2018 구글 트렌드 한국 1위 진입.
- **동물상 AI 분류기 등장**(2019, 조코딩 등) — "검사 결과 = 텍스트"에서 "검사 결과 = 이미지 라벨"로 이동한 분기.

### 2020~2022 — 폭발기 (코로나 + 매스미디어 발화)
- **2020.06.13 '놀면 뭐하니?' 방영 → 6월 셋째주 검색량 정점**(디지털 인사이트). TV 단일 트리거로 한국 MBTI 대중화 결정적 분기점.
- **푸망 2020.12 런칭, 방구석연구소 2020년 자리잡음** — 짤테스트 양산 플랫폼 본격 등장.
- **꽃 MBTI 테스트(LU42, 2020.11)**: 13문항 → 꽃 결과 + 궁합 + 공유 시 의료진 기부 1,000원. SNS 폭발.
- **호구 성향 테스트(국민약속, 2020.11)**: 8유형(흑우/흑미어캣 등). **출시 후 누적 140만+ 응시**.
- **퍼스널컬러 자가진단**(ktestone, Withsome 2021): 12문항 → 컬러 결과. **Withsome 글로벌 TikTok 3,000만+ 플레이**, 한국발 컨텐츠 글로벌 역수출 시발점.
- **ktestone "어떤 OOO인가요" 시리즈**: 일러스트 + 짧은 결과로 카톡 공유 표준 포맷 정착.

### 2022~2024 — 양산기 + 사회 침투
- 방구석연구소·푸망 자체 IP 짤테스트 양산("연애 스킬", "아이돌 포지션", "베테랑", "탕진력" 등).
- 사주+MBTI 결합(MBTIID, 사주MBTI, 타이트사주) — 연애 시장 호환성 분석 트렌드.
- 캐릭터 매칭(왁타버스·호요버스 캐릭터 MBTI) — IP 팬덤 결합.
- **MBTI 사회 인프라화**: 자기소개 표준, 채용 자소서 기입 요구(SH수협은행·아워홈·LS전선 2022), 은행 앱 푸시 알림까지 침투. **차별·고정관념 backlash 누적**("INFP는 면접도 못 보나요?" 여성신문·한국일보).

### 2024~2026 — 에겐/테토 + MBTI 피로 균열
- **에겐·테토 테스트**: 2024.3 인스타툰 작가 "내쪼" 게시물 → 2025 중반 폭발. 호르몬(테스토스테론/에스트로겐) 기반 4유형(테토남/에겐남/테토녀/에겐녀). **2025.7 기준 누적 100만+**, 푸망·메타브·타입스 멀티 플랫폼 동시 호스팅. 대기업(고려아연) 콜라보까지.
- **'Are you a T?' 밈**: MBTI T(사고형) 응답을 공감 부족 비난용으로 쓰는 신조어(Korea Herald 2024).
- **"요즘은 MBTI 안 해요"** 헤드라인(매경 2025-05) — MBTI 피로감 + 식상함 균열 확인.
- 퍼스널컬러가 **NYT 보도(2024)로 미국 역수출** — 한국 컨텐츠의 글로벌 침투 본격화.

## 5. 아시아 비교 — 한국 특수성 확인

### 일본
- 전통적으로 **혈액형 성격설(ketsueki-gata)**이 짤테스트 자리 점유(1927 후루카와 다케지). 일본인 절반+ 혈액형 성격 대화 일상.
- MBTI/16personalities는 **K-pop·BTS·한류 매개**로 청년층에 침투. 한국발 도착 시차 명확. 애니메이션 캐릭터 MBTI 부여 fan culture 성장 중.

### 중국
- 2022~2023 Weibo **MBTI 해시태그 35억 뷰, 80만 명 토론**. 이력서 MBTI 기입 일상화.
- **2026.4 SBTI(Silly Big Type Indicator) 패러디 폭발** — 빌리빌리 발 자조형, WeChat 지수 4,085만, Weibo·샤오홍슈 2,000만+ 토론. "monk, mum, solo, shit, sexy, dead" 27종 자조 라벨. 한국 짤테스트 톤과 유사 — **한국보다 1~2년 늦은 사이클**.

### 대만·동남아
- **Taiwan Design Expo 2023 Innate Personality Traits 테스트** — 12 메타포 질문 + 16개 원소 결과(빛/돌/철/요정/고양이). 싱가포르·동남아 확산. 디자인 완성도가 viral 동력 — **한국 짤테스트 공식과 동일**.
- 베트남·태국: 자체 플랫폼 빈약, 16personalities·uQuiz + 한국 짤테스트 번역본 소비.

### 한국 vs 글로벌 차이 요약
1. **한국 = MBTI를 사회 인프라로 사용** (소개·소개팅·자소서·채용·은행 푸시). 영어권은 retreats 워크숍 도구 정도.
2. **한국 = 사물·이미지 매칭 비중 큼** (동물상·꽃·색·과자). 영어권 = IP 캐릭터 매칭 비중 큼.
3. **한국 = 시각 결과 페이지 산업화** (푸망·방구석·테스트모아 등 전문 플랫폼). 일본은 혈액형, 미국은 BuzzFeed로 충분.
4. **한국 = 퍼스널컬러 종주국**(NYT 2024).
5. **한국 = MBTI + 사주 동시 소비**(포스텔러 이용자 83% MZ). 서구는 라이트·헤비 분리.
6. **한국 = 공중파 방송 단일 트리거 효과**(2020.06.13). 영어권은 이런 사례 드뭄.

## 6. 비즈니스 모델 (다음 단계 입력용)

### 한국 짤테스트 플랫폼 수익 구조
- **푸망 모델**: 자사 테스트로 트래픽 쌓고 기업 IP 콜라보 영업. **AE 직군**이 매출 핵심.
- **방구석연구소 모델**: 자체 미디어 + 데이터 마케팅 솔루션 결합. "마케팅 컨설팅 + 콘텐츠 제작" 패키지.
- **ktestone 모델**: 개인 운영 + 광고 + 사주 결제 추정. 1인 모델로는 푸망 규모 도달 불가.
- 짤테스트 콜라보 단가: **테스트 1건당 수천만~억 단위 추정**(트래픽·IP 가치에 따라). 정확한 수치는 비공개.

### 라이프사이클 패턴
- **단명 사례**: LU42 꽃 테스트(캠페인 종료 후 소멸), 호구테스트(도메인 살아있으나 신규 갱신 없음) → 1회성 캠페인·신규 콘텐츠 없음.
- **장수 사례(5년+)**: 푸망·방구석연구소·ktestone·16personalities·BuzzFeed → 공통적으로 **신규 테스트 지속 발행**으로 플랫폼화.
- 결론: 짤테스트는 **단일 테스트 = 단명, 플랫폼 = 장수**. 단일 "공감 MBTI" 1발만으론 1년 안에 죽음.

## 7. 수집 중 발견한 "왜 흥했나" 단서 (02~05단계 입력)

1. **트리거 사건의 힘** — 2020.06.13 '놀면 뭐하니' 단일 방송이 검색량 정점 만듦. TV·인플루언서가 짤테스트 트래픽 도화선.
2. **시각 패키징이 본질** — 16personalities 캐릭터 일러스트 + 강점/약점/연애/커리어 카드 구조 = 한국 짤테스트 표준 포맷. 텍스트 결과는 안 퍼짐.
3. **공유 메커닉이 컨텐츠보다 중요** — BuzzFeed 데이터: 퀴즈 페북 공유 일반 기사의 3.3배. **결과 캡처의 인스타 스토리 친화도**가 짤테스트 핵심 KPI.
4. **불안의 시대 = 자기 라벨링의 시대** — 팬데믹 격리·취업난·결혼 회피 흐름에서 "나는 누구인가" 답을 손쉽게 주는 컨텐츠 폭발.
5. **MBTI 피로감 → 차세대 컨텐츠 기회** — "요즘은 MBTI 안 해요" 헤드라인(매경 2025-05). 신뢰성 비판·차별 우려·식상함이 누적. **공감 품질 + 새로운 축**으로 들어가는 게 빈자리.
6. **사주+MBTI 융합** — 데이팅·결혼 시장에서 "사주만으론 부족, 데이터로 호환성" 욕구. 결과 페이지에 **궁합 표**가 트래픽 무기.
7. **한국형 정식 검사 토양** — KEPTI·k-DISC·MMPI-2 한국판 등 "한국인 데이터로 표준화"는 학술적 정당성 마케팅 자산. **정식 심리학 외양 + 짤테스트 UX 하이브리드**가 차별점.
8. **차별·고정관념 backlash** — 채용 차별, 성별 이분법 비판 누적. 결과 페이지에서 **"이 유형은 이런 사람" 단정 회피**, **"이런 상황에서 이렇게 느낄 가능성"** 카피가 차별점 가능.
9. **마케팅 테스트 피로 균열** — Reddit +148 "은행 앱이 내 MBTI 물어봄". 진정성 있는 컨텐츠가 비집고 들어갈 틈 존재.
10. **한국 → 글로벌 수출 루트 존재** — Withsome 컬러 테스트 글로벌 TikTok 3,000만+. 한국 시장 fit 후 영어 버전 확장 검토 가능.
11. **단일 테스트 단명 → 플랫폼화 필수** — 같은 IP 위에 **신규 테스트 시리즈** 발행 구조를 처음부터 설계해야 1년 후 살아남음.

## 8. 이 리서치의 맹점

- **트래픽 정확치 부족** — 푸망·방구석연구소의 정확한 MAU/DAU·매출은 비공개. 추정치 위주.
- **짤테스트 콜라보 광고 단가 비공개** — B2B 영업 가격대는 업계 내부 정보, 외부 인용 없음.
- **개인 운영 짤테스트 사이트 다수 누락** — ktestone 외에도 1인 개발자 사이트 다수 존재하지만 카탈로그 한계.
- **시간 민감성** — 에겐테토(2025~) 등 최근 트렌드는 6개월 내 변동 가능성 큼. **2026.05 시점 스냅샷**.
- **학술 분석 부족** — 한국 KCI 논문에서 짤테스트 분석은 일부 확인됐으나 전수 조사 안 함. 02~03단계에서 보강 필요.
- **유튜브 심리 컨텐츠 채널은 가볍게만 다룸** — 뇌부자들 등 채널 분석은 별도 단계 필요시 확장.
