## 비자명 제약

- 토스 디자인을 **그대로 복제하기 위함이 아니다.** 토스가 "공감 / 한국식 / 모바일 / 공유 친화"라는 우리 목표와 같은 방향을 이미 풀어놨으니, 그 풀이를 학습 자료로 본다.
- 결과지 컴포넌트 마크업에 토스 스타일을 직접 박지 말 것 — 원칙만 가져오고 구현은 우리 톤으로.
