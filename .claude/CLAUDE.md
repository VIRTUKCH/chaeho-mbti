# .claude

**책임:** Claude Code 전용 컨텍스트·문서. 사용자가 직접 읽기 위한 게 아니라 AI가 작업할 때 로드되는 자리.

## 진입점

- [docs/](docs/) — 영구 가이드 (컨텍스트 엔지니어링, 심리학, 디자인, 바이럴).
- [ai-context/](ai-context/) — 세션 간 누적되는 소모성 메모 (현재 비어있음).

## 비자명 제약

- 여기 들어가는 모든 문서는 **AI 작업 참조용**. 사용자용 README가 아니므로 결론·룰 중심으로 짧게.
- `docs/` 와 `ai-context/` 의 차이: docs = 영구 가이드, ai-context = 소모성. 영구 가치 있으면 docs로 승격.
